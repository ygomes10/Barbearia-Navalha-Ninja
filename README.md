# 💈 Barbearia Navalha Ninja

Uma landing page moderna e responsiva desenvolvida para a **Barbearia Navalha Ninja**, focada em proporcionar uma experiência visual marcante e facilitar o agendamento de clientes através do WhatsApp.

## 🌐 Acesse o projeto

🔗 **Site:** https://barbearianavalhaninja.netlify.app/

🔗 **Repositório:** https://github.com/ygomes10/Barbearia-Navalha-Ninja

---

## 📖 Sobre o projeto

O objetivo deste projeto foi desenvolver uma landing page profissional para uma barbearia, utilizando apenas tecnologias web nativas, priorizando:

- Design moderno;
- Performance;
- Responsividade;
- Experiência do usuário (UX);
- SEO básico;
- Código organizado e de fácil manutenção.

---

## ✨ Funcionalidades

- 💈 Apresentação da barbearia
- 📱 Layout totalmente responsivo
- 🖼️ Galeria de cortes em formato de slider
- 🔍 Lightbox para visualização ampliada das imagens
- 📲 Botões de agendamento integrados ao WhatsApp
- 🎯 Navegação com rolagem suave
- ⚡ Imagens otimizadas com `loading="lazy"`
- 🔍 Estrutura preparada para indexação no Google

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts
- Netlify

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/ygomes10/Barbearia-Navalha-Ninja.git
```

Acesse a pasta:

```bash
cd Barbearia-Navalha-Ninja
```

Abra o arquivo `index.html` no navegador ou utilize a extensão **Live Server** do Visual Studio Code.
